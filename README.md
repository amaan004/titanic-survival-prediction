# Titanic Survival Prediction 🚢

A complete ML pipeline built on the Titanic dataset — 
from raw data to a trained Random Forest classifier.

## What I did
- Cleaned and explored the dataset using Pandas
- Identified key survival factors through EDA (Gender, Fare, Pclass)
- Engineered features (FamilySize, Sex encoding)
- Trained a Random Forest classifier achieving **78% accuracy**
- Evaluated using confusion matrix and feature importance analysis

## Key Findings
- Female passengers had significantly higher survival rates
- Passenger class and fare were strong predictors of survival
- Adding the Sex feature improved accuracy from 69% → 78%
